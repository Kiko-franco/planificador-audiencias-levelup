# levelUp Audience Planner (web interna)

MVP listo para GitHub + Vercel.

## Ejecutar en local
1) Instala Node.js (18+)
2) En la carpeta del proyecto:
```bash
npm install
npm run dev
```

## Deploy en Vercel
1) Sube esta carpeta a GitHub (todo el contenido)
2) En Vercel: New Project → Import repo → Deploy

## Logo
El logo se sirve desde: `public/levelup-logo.png`
